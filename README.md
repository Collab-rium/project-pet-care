# Pet Care - MVP

Manage pets with reminders for meals, medicines, baths, vet visits.

---

## Quick Start

```bash
# Terminal 1: Backend
cd backend && npm start

# Terminal 2: Frontend
cd frontend && flutter run
```

Done. App is running.
