### Dynamic Themes (MVP Enhancement)

**Description**:
- The app's theme dynamically adapts based on the pet's type or uploaded image.
  - Example: If the pet is a tree or plant, the theme adopts a green/forest aesthetic.
  - If an image of a dog or cat is uploaded, the app's theme aligns with the predominant image mood or color palette.

**Approach**:
1. **Simplified Method** (Basic MVP):
   - Use color-detection libraries to identify predominant colors in the uploaded image.
   - Dynamically adjust the app's primary/secondary UI colors.

2. **Advanced Method** (Stretch Goal):
   - Pre-map themes based on pet types (dog, cat, bird, plants) for a stronger user-defined experience.
   - Allow users to manually adjust themes after detection.

The dynamic theme adds a personalized touch and makes the app visually appealing to users while still being lightweight and cost-free.
